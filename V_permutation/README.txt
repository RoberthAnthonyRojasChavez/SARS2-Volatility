To run the code, type "python implement.py" in the console while the current directory is set to the (unzipped) folder.

It asks for teh following arguments:

	Position of interest. (eg. 14)
	Neighborhood Size. (10)
	Number of permutations. (1000)

The code outputs:
	List of Neighbors associated with the position of interest.
	Null Statistic
	p-value
